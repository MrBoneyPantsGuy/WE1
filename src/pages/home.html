Web Engineering - Solution-Navigator

Anleitung

  1. Durch einen Klick auf einen Punkt innerhalb des oberen Menüs wird ein Untermenü mit den einzelnen Aufgaben/Dateien angezeigt.
  2. Durch einen Klick auf eine Datei innerhalb des Untermenüs wird diese in der Quellcodeansicht dargestellt.
  3. In der Beschreibung zur jeweiligen Aufgabe, kann über den Button "In Aktion anzeigen!" die Lösung als HTML/Javascript-Code ausgeführt und angezeigt werden.
  4. Falls der anzuzeigende Quellcode zu lang ist, kann der "fehlende" Inhalt über die Scrollbar eingesehen werden.
